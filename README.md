BingRewards
===========

Crontab:
0 13 * * * sleep $[RANDOM \% 120]m ; python2 /home/nathan/bingrewards/BingRewards/main.py 2>&1 | gzip > /home/nathan/bingrewards/var/log/`date "+\%Y-\%m-\%dT\%H:\%M:\%S"`.log.gz
